# @armin/sync-server

Placeholder for Armin's optional, paid cloud sync service.

The service stores a Profile's data as opaque, end-to-end-encrypted bytes — it
never parses the study domain (see `CONTEXT-MAP.md`). Nothing is implemented yet;
this workspace exists to stake out the boundary.

When work starts here, add `CONTEXT.md` and `docs/adr/` for the sync context.
