# Armin

Armin is a local-first study context for retaining knowledge through cards,
decks, reviews, and prerequisite relationships.

## Language

**Deck**:
A named collection of cards studied together.
_Avoid_: Course, folder, set

**Card**:
An atomic study item with a prompt and an answer.
_Avoid_: Flashcard when naming code concepts

**Prerequisite**:
A card that must be learned before another card should be studied.
_Avoid_: Parent, dependency

**Dependent card**:
A card that relies on one or more prerequisite cards.
_Avoid_: Child, subcard

**Prerequisite graph**:
The directed knowledge structure formed by prerequisite relationships between
cards.
_Avoid_: Tree

**Locked card**:
A card that is not yet ready to study because at least one prerequisite has not
been learned.
_Avoid_: Disabled card, blocked card
